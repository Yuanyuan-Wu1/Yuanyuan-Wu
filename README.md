## How To Use
1. Clone the project
```bash
git clone https://github.com/Yuanyuan-Wu1/portfolio-website.git
```

2. Go to the project directory
```bash
cd portfolio-website
```

3. Install dependencies
```bash
npm install
# or
yarn
```

4. Start the development server
```bash
npm run dev
# or
yarn dev
```
